# Требования для ниши: Промышленное оборудование / B2B-производство

Добавляются поверх universal.md. Длинный цикл сделки, ЛПР — директора производств, главные инженеры/механики, снабжение.

## Несколько направлений — описывать раздельно
Производитель часто ведёт 2+ направления (железо / интеграция софта / собственное ПО / сервис). Это РАЗНЫЕ ниши — описывать и позиционировать раздельно. Объединять в «единый комплекс/экосистему» — только при подтверждении, иначе [ГИПОТЕЗА].

## Каналы-кандидаты (оценить ВСЕ, каждый — рекомендуем/не сейчас+почему)
- **Яндекс.Директ (поиск + РСЯ)** — горячий спрос ЛПР («купить <оборудование>», «<оборудование> цена»). CPL B2B ориентир 3 000–8 000 ₽.
- **SEO (Яндекс)** — дешёвые лиды через 4–6 мес; кластеры по типам оборудования и применениям. CPL в разы ниже Директа.
- **Авито (и профильные маркетплейсы/классифайды)** — ПЕРВИЧНЫЙ для B2B-оборудования: ЛПР ищут оборудование на Авито; есть Авито PRO для бизнеса. Не занижать.
- **2ГИС + Яндекс.Карты** — проверяют поставщика перед обращением; локальный охват региона производства.
- **Telegram** — экспертный контент + посев в отраслевых чатах («Металлообработка», «Промавтоматизация»); B2B-ЛПР активны.
- **Email-рассылка по базе** — повторные продажи: расходники, сервис, апсейл.
- **Отраслевые порталы, тендерные площадки, выставки** — целевой, но длинный цикл; подключать после онлайн-воронки.
- **VK** — обычно слабый для ЛПР заводов; чаще «не сейчас» (но проверять по нише).

## Используемые vs рекомендуемые
Отдельно фиксировать, что клиент УЖЕ использует (Директ/Авито/…) — только при подтверждении (признак с URL или подтверждение клиента). Нет данных → [НЕДОСТАТОЧНО ДАННЫХ], но рекомендацию канала это не отменяет.

## Соцдоказательства (искать про самого клиента)
Крупные клиенты/референсы, реализованные проекты-кейсы, отзывы и рейтинг (сайт компании, Яндекс.Карты, 2ГИС, Otzovik). Для B2B доверие решает — выносить в позиционирование и кейсы.

## Специфика целевой аудитории
- Снабжение / закупки: цена, сроки поставки, гарантия, наличие.
- Технические ЛПР (гл. инженер/механик): соответствие ТЗ, интеграция, сервис и запчасти.
- Топ-менеджмент: окупаемость, импортозамещение, надёжность поставщика.

## KPI отрасли (нормы для сравнения)
- Цикл сделки: 1–8 мес (зависит от суммы и направления).
- CPL Директ B2B: 3 000–8 000 ₽; SEO: в разы ниже при выходе в топ.
- Конверсия сайта B2B-производителя: 1–3% (квалифицированная заявка).

## AI-инструменты специфичные для ниши
- Автоподбор/конфигуратор оборудования по параметрам клиента (квалификация заявки).
- AI-генерация технических кейсов и SEO-статей по реализованным проектам.
- Авто-оповещение по статусу заказа/поставки/сервиса.
- AI-квалификация входящих (тип производства, объём, срок) → передача менеджеру.

## Обязательные AI-рекомендации (полный И краткий отчёт)
Поверх универсальной контент-автоматизации:
- **AI-квалификация входящих заявок** (чат-бот: 3–5 вопросов → карточка в CRM) — менеджер тратит время только на горячих.
- **Авто-оповещение клиента о статусе заказа/поставки** — снижает «что по моему заказу?», растит доверие.
