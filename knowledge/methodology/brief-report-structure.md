# Краткий отчёт для клиента — структура

**Целевая аудитория:** владелец бизнеса. Время на чтение: 5 минут.
**Формат:** интерактивный HTML в приложении + скачать PDF.

## Жёсткие правила

- Только **факты** с RS ≥ 3. Гипотезы → в полный отчёт
- Все суммы в ₽
- На каждый раздел: 1 вывод + 3 факта + 1 действие (Minto pyramid)
- AI-секция — значимая (не сноска), 3 главных предложения
- Релевантные каналы РФ только (см. `channels/russia-channels-map.md`)
- Каждый факт со ссылкой на источник по hover/click

## Секции

### 0. Шапка
- Название компании
- Ниша
- Дата
- Методология: Minto · Knaflic · Duarte
- **Big idea (Duarte):** одно ключевое сообщение крупно — «главное окно роста за 3 месяца»

### 1. Бизнес
- **ВЫВОД** (одно предложение)
- 3 факта с RS
- **ДЕЙСТВИЕ** этой недели (0 ₽ или минимум бюджета)

### 2. Рынок
- ВЫВОД → 3 факта → ДЕЙСТВИЕ

### 3. Целевая аудитория
- ВЫВОД → 3 факта → ДЕЙСТВИЕ

### 4. Каналы (только релевантные)
- ВЫВОД → 3 факта о текущих каналах → ДЕЙСТВИЕ
- Сравнительная инфографика: каналы × эффективность × бюджет

### 5. Конкуренты
- ВЫВОД о свободной позиции (Ries & Trout)
- Сравнительная таблица 3–5 конкурентов (только подтверждённые факты)
- ДЕЙСТВИЕ — занять свободную ментальную ячейку

### 6. AI-автоматизация ⭐ (значимый раздел)
- 3 главных AI-предложения в формате Duarte-контраст:
  - 🔴 Сейчас → 🤖 Решение → 🟢 После → 💰 Эффект
- Каждое — раскрывается по клику для деталей
- См. шаблон в `ai-automation-block.md`

### 7. Стратегия 3 месяца
- 🔴 Эта неделя (0 ₽) — список действий
- 🟡 Этот месяц (бюджет ~XX тыс. ₽)
- 🟢 3 месяца (бюджет ~XX тыс. ₽)
- Каждый пункт — с RS-оценкой и связью с фактом из секций выше

### 8. STAR-факт (Duarte)
- Одна цифра/цитата на финал, которая запомнится клиенту

### 9. CTA
- Кнопка «Скачать PDF» — упрощённая версия с теми же фактами
- Кнопка «Связаться» — для апселла (доп. услуги по внедрению AI)

## Визуальные правила (Knaflic)

- Один акцентный цвет на страницу
- 🔴 для проблем, 🟢 для возможностей, 🟡 для нейтральных
- Цифры — крупно, контекст — мелко
- Никаких 3D-графиков
- Инфографика — где данных ≥ 3 точек
