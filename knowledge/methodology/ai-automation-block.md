# Блок AI-автоматизации — методология

Этот блок — **сердце нашего апселла**. Клиент видит конкретный путь от проблемы к выгоде с цифрами, мы получаем продажу услуги по внедрению.

## Шаблон одной AI-возможности (Duarte-контраст)

```markdown
### <Название идеи>

🔴 **СЕЙЧАС:** <текущее состояние с цифрами — часы, ₽, конверсия>
🤖 **РЕШЕНИЕ:** <как работает AI-инструмент в одном предложении>
🟢 **ПОСЛЕ:** <новое состояние с цифрами>
💰 **ЭФФЕКТ:** <экономия ч/мес + ₽/мес ИЛИ прирост лидов/выручки>
📡 **КАНАЛЫ:** <где применять>
🛠️ **ИНСТРУМЕНТЫ:** <конкретные API/сервисы>
⏱️ **ВНЕДРЕНИЕ:** <разовая ₽ + ежемесячная ₽>
🎯 **СРОК ОКУПАЕМОСТИ:** <месяцев>
```

## Правила построения AI-предложений

### 1. Идея должна быть привязана к ФАКТУ
Нельзя предлагать чат-бот для квалификации лидов, если в фактах нет данных о времени менеджеров на первичный контакт. Каждое AI-предложение → ссылка на факт-источник.

### 2. Цифры экономии — конкретные
- ❌ «Сэкономит много времени»
- ✅ «30–50 ч/мес юриста = 18 750–31 250 ₽/мес при ставке 100 тыс. ₽/мес»

### 3. Контраст обязателен (Duarte)
Без сравнения «сейчас vs после» — это не AI-блок, это просто список инструментов.

### 4. Стоимость внедрения честная
Если знаем точно — пишем. Если оценка — пометка [ОЦЕНКА].

### 5. Релевантность нише
Для юр.услуг — оповещение клиентов о ходе дел. Для e-commerce — рекомендательные системы. Не путать.

## Краткий отчёт vs полный отчёт — что показывать

### Краткий (клиент)
- **3 главных идеи** (Knaflic — declutter)
- Самые очевидные ROI
- Готовые к запуску
- С интерактивом — клик раскрывает детали

### Полный (владельцы приложения)
- **5–10 идей** включая экспериментальные
- С пометкой [PoC] для непроверенных
- С детальным анализом стоимости и рисков
- Все, что обнаружили в исследовании

## Источники AI-идей

1. **niches/_all.md** — универсальные (контент, рассылки, чат-боты)
2. **niches/<niche>.md** — нишевые
3. **Конкретные факты из исследования** — что у клиента болит

## Антипаттерны

- ❌ «AI поможет улучшить маркетинг» — пустота
- ❌ Идеи без цифр — нерелевантны
- ❌ Копипаст из других ниш — клиент почувствует фальшь
- ❌ Универсальные «волшебные» решения — не работают
