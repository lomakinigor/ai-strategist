# Правила достоверности данных

## Reliability Score (RS)

| RS | Описание |
|----|----------|
| 5  | Официальные реестры (ЕГРЮЛ, Росстат, ФГИС) и официальные документы |
| 4  | Официальный сайт, официальные карточки, качественные деловые источники (РБК, Ведомости, Коммерсант) |
| 3  | Официальные соцсети компании, вакансии, карты, публичные Telegram-каналы компании |
| 2  | Реклама, косвенные сигналы, пользовательские отзывы, агрегаторы отзывов |
| 1  | Неверифицированные агрегаторы — **не использовать как основание стратегии** |

## Логика уверенности

Каждый факт или вывод в системе сопровождается:
- **источник** — URL или название источника
- **дата** — дата получения или публикации
- **RS** — Reliability Score (1–5)

Уровень достоверности вывода определяется минимальным RS среди опорных фактов:
- **HIGH** — все опорные факты RS ≥ 4
- **MEDIUM** — есть факты RS 3, нет RS ≤ 1
- **LOW** — есть факты RS 2 или вывод строится на малом числе источников

## Классификация утверждений

### ФАКТ
Утверждение подтверждено источником RS ≥ 3. Источник указан явно.

```
ФАКТ [RS:4] Выручка компании в 2023 году составила X руб.
Источник: официальный сайт, дата: 2024-01-15
```

### ГИПОТЕЗА
Утверждение вероятно, но не подтверждено напрямую. Основано на косвенных сигналах (RS ≤ 2) или логическом выводе из ФАКТОВ.

```
ГИПОТЕЗА [MEDIUM] Компания, вероятно, работает в B2B-сегменте.
Основание: характер вакансий (RS:3), отсутствие B2C-рекламы (RS:2)
```

### НЕДОСТАТОЧНО ДАННЫХ
Данных для вывода нет или они ненадёжны (RS ≤ 1). Вывод не делается.

```
НЕДОСТАТОЧНО ДАННЫХ: финансовые показатели компании не раскрыты публично.
Рекомендация: запросить у клиента.
```

## Запрет на неподтверждённые выводы

- LLM не делает выводов на основе внутренних "знаний" о конкретной компании
- Мировая аналитика применяется только к анализу ниши/рынка и всегда с пометкой:
  > «Мировая аналитика — применимость к российскому рынку не гарантирована»
- Если данных недостаточно — явно указывать «НЕДОСТАТОЧНО ДАННЫХ», а не додумывать
- Суммы в стратегиях и рекомендациях — только в рублях
